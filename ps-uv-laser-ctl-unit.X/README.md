# MEP制御BOX コマンド仕様

## 通信概要

PCからPICマイコンへコマンドを送信し、PICマイコンからPCへ実行結果または状態を返します。

### 制御文字

| 表記 | 16進数 | 意味 |
|---|---:|---|
| `<HT>` | `0x09` | 水平タブ |
| `<LF>` | `0x0A` | ラインフィード |
| `<CR>` | `0x0D` | キャリッジリターン |
| `<SPC>` | `0x20` | スペース |

## 共通フォーマット

### PC → PIC

- ヘッダー: なし
- フッター: `<CR>`

```text
COMMAND<CR>
```

数値パラメータを持つコマンドは、コマンド名と値を `/` で区切ります。

```text
COMMAND/VALUE<CR>
```

### PIC → PC

- ヘッダー: `C<HT>`
- フッター: `<CR><LF>`

```text
C<HT>RESPONSE<CR><LF>
```

## コマンド一覧

| 送信コマンド<br>PC → PIC | 戻り値<br>PIC → PC | 内容 | 備考 | 送受信例 |
|---|---|---|---|---|
| `VER` | `VERSION<SPC>X.X` | FWバージョン問い合わせ | `X.X`: FWバージョン | `VER<CR>` → `C<HT>VERSION<SPC>1.0<CR><LF>` |
| `HAL` | `HAL<HT>X` | ホール素子状態問い合わせ | `X=0`: ホール素子Hi、シャッターオープン<br>`X=1`: ホール素子Lo、シャッタークローズ | `HAL<CR>` → `C<HT>HAL<HT>1<CR><LF>` |
| `SHT/X` | `SHT<HT>X<HT>HAL<HT>Y` | シャッター設定 + ホール素子状態問い合わせ | `X=0`: シャッタークローズ<br>`X=1`: シャッターオープン<br>`Y=0`: ホール素子Hi<br>`Y=1`: ホール素子Lo | `SHT/0<CR>` → `C<HT>SHT<HT>0<HT>HAL<HT>1<CR><LF>` |
| `COA/XXX` | `COA<HT>YYY` | 同軸照明明るさ設定 | `XXX=0〜255`: 明るさ設定値<br>`YYY=30〜318`: マイコン内部の換算値 | `COA/50<CR>` → `C<HT>COA<HT>78<CR><LF>` |
| `RG1/XXX` | `RG1<HT>YYY` | リング照明1 明るさ設定 | `XXX=0〜255`: 明るさ設定値<br>`YYY=0〜199`: マイコン内部の換算値 | 例: `RG1/255<CR>` → `C<HT>RG1<HT>199<CR><LF>` |
| `RG2/XXX` | `RG2<HT>YYY` | リング照明2 明るさ設定 | `XXX=0〜255`: 明るさ設定値<br>`YYY=0〜199`: マイコン内部の換算値 | 例: `RG2/255<CR>` → `C<HT>RG2<HT>199<CR><LF>` |
| `RG3/XXX` | `RG3<HT>YYY` | リング照明3 明るさ設定 | `XXX=0〜255`: 明るさ設定値<br>`YYY=0〜199`: マイコン内部の換算値 | 例: `RG3/255<CR>` → `C<HT>RG3<HT>199<CR><LF>` |
| `RG4/XXX` | `RG4<HT>YYY` | リング照明4 明るさ設定 | `XXX=0〜255`: 明るさ設定値<br>`YYY=0〜199`: マイコン内部の換算値 | 例: `RG4/255<CR>` → `C<HT>RG4<HT>199<CR><LF>` |
| `LDP/X` | `LDP<HT>X` | レッドレーザー設定 | `X=0`: レッドレーザーOFF<br>`X=1`: レッドレーザーON | 例: `LDP/1<CR>` → `C<HT>LDP<HT>1<CR><LF>` |

## コマンド詳細

### `VER`

FWバージョンを問い合わせます。

送信:

```text
VER<CR>
```

応答:
```text
C<HT>VERSION<SPC>1.0<CR><LF>
```

### `HAL`
メカニカルシャッターに取り付けたホール素子の状態を問い合わせます。

送信:

```text
HAL<CR>
```

応答:

```text
C<HT>HAL<HT>X<CR><LF>
```

| 値 | 意味 |
|---:|---|
| `0` | ホール素子Hi、シャッターオープン |
| `1` | ホール素子Lo、シャッタークローズ |

### `SHT/X`

シャッターを開閉し、動作後のホール素子状態を返します。

送信:
```text
SHT/X<CR>
```

応答:
```text
C<HT>SHT<HT>X<HT>HAL<HT>Y<CR><LF>
```

### `COA/XXX`

同軸照明の明るさを設定します。

送信:

```text
COA/XXX<CR>
```

応答:

```text
C<HT>COA<HT>YYY<CR><LF>
```

| 値 | 範囲 | 内容 |
|---|---:|---|
| `XXX` | `0〜255` | PCから送信する明るさ設定値 |
| `YYY` | `30〜318` | マイコン内部の換算値 |

### `RG1/XXX`、`RG2/XXX`、`RG3/XXX`、`RG4/XXX`

リング照明の明るさを設定します。

送信:

```text
RGn/XXX<CR>
```

- `n`: `1〜4`
- `XXX`: `0〜255`

応答:

```text
C<HT>RGn<HT>YYY<CR><LF>
```

| 値 | 範囲 | 内容 |
|---|---:|---|
| `XXX` | `0〜255` | PCから送信する明るさ設定値 |
| `YYY` | `0〜199` | マイコン内部の換算値 |

### `LDP/X`

レッドレーザーのON/OFFを設定します。

送信:

```text
LDP/X<CR>
```

| X | 動作 |
|---:|---|
| `0` | レッドレーザーOFF |
| `1` | レッドレーザーON |

応答:

```text
C<HT>LDP<HT>X<CR><LF>
```
